# 💼 PayrollPro - Employee Payroll Management System

PayrollPro is a web-based Employee Payroll Management System developed using Angular.  
It helps organizations manage employees, calculate salaries, and generate payslips easily.

---

## 📌 Features

✔ Secure Login System  
✔ Dashboard with Statistics & Charts  
✔ Employee Management (Add / View / Delete)  
✔ Salary Calculator  
✔ Payslip Generator  
✔ Responsive UI (Mobile / Desktop)  
✔ Professional Glass UI Design  
✔ Angular Based Frontend  

---

## 🛠️ Technologies Used

- Frontend: Angular 15+
- Language: TypeScript, HTML, CSS
- Styling: Bootstrap + Custom CSS
- Charts: Chart.js
- Backend: (Optional / API Based)
- Database: (Optional / API Based)
